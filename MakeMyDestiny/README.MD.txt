# Travel Booking System

This is a basic travel booking web application built using Flask.

## Features
- User Signup & Login
- Ticket Booking
- Seat Selection
- Admin Dashboard

## Tech Used
- Python (Flask)
- SQLite
- HTML, CSS, JavaScript

## How to Run
1. Install Flask
2. Run app.py
3. Open http://127.0.0.1:5000

## Note
This is a beginner-level project.
