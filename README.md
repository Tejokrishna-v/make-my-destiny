# 🎟️ Make My Destiny

Make My Destiny is a console-based travel booking system built using Python.  
It allows users to create accounts, log in, and book travel tickets such as Bus, Train, Flight, and Hotel.

---

## 🚀 Features

- User Account Creation
- OTP Verification System
- Login Authentication
- Bus Booking
- Train Booking
- Flight Booking
- Hotel Booking
- Seat Selection
- Multiple Payment Options

---

## 🛠️ Technologies Used

- Python
- Random Module
- Time Module
- Dictionary Data Structure
